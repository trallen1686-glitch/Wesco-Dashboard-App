Client ID Setup Help

  1. Go to portal.azure.com and sign in with your Microsoft work account.
  2. Search for Microsoft Entra ID.
  3. Open App registrations.
  4. Click New registration.
  5. Name it Wesco Master Subcontractor Calendar.
  6. Choose Accounts in this organizational directory only if only Wesco users will use it.
  7. Under Redirect URI choose Single page application and paste the URL where this HTML will run.
  8. After it is created copy the Application (client) ID.
  9. Paste that value into this line in the HTML file:
clientId: "cb882f2c-7555-427e-8484-92d6d0a21494"
Then go to API permissions and add User.Read and Calendars.ReadWrite. If needed, ask your Microsoft admin to grant admin consent.
