About this screen

This screen signs in with Microsoft and reads live data from the real Documents / Projects folder in Wesco's SharePoint through Microsoft Graph — nothing here is hardcoded, so new, renamed, or moved project folders show up automatically the next time you open a category.

Click a category on the left, then click any folder to open it right here on the screen — use the breadcrumb at the top to go back up. Clicking a file (like a tracker spreadsheet) opens it in SharePoint in a new tab, since files still need Word/Excel/SharePoint to view.

If sign-in fails, double check the Redirect URI registered on the Wesco Dashboard App registration in Azure matches this page's exact web address, and that Files.Read.All / Sites.Read.All permissions have admin consent granted.
